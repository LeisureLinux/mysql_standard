# mysql_standard
标准化 MySQL 数据库
在日常 MySQL 数据库运维中，比较头疼的除了字符集以外，还有就是表的引擎之类的，都没有统一起来。
这个脚本，把所有的 MyISAM 引擎的表，都修改为 InnoDB 。
另外，字符集的选择上，很多菜鸟都用 utf8，而没有明白 utf8 和 utf8mb4 的区别。
为了更好的兼容中文，把所有原先的配置的 utf8 字符集的表以及列，都修改为 utf8mb4 。
